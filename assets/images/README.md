# Images & Logo Folder

Add your app logo and images here. Supported formats:
- `.png` (recommended for logos with transparency)
- `.jpg` / `.jpeg`
- `.svg` (with flutter_svg package)

## Usage

After adding images to this folder, reference them in Dart code as:

```dart
Image.asset('assets/images/synapse_logo.png')
```

This folder is already configured in `pubspec.yaml` to be bundled with your app.
